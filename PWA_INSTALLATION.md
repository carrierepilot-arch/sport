# 📱 Installation de l'app Sport sur Android

L'application Sport est maintenant disponible en tant que **Progressive Web App (PWA)** qui peut être installée sur votre téléphone Android comme une véritable application native.

## ✨ Avantages de la PWA

- ✅ Accès rapide depuis l'écran d'accueil
- ✅ Fonctionne en mode hors ligne (même données que lors de votre dernière visite)
- ✅ Pas d'appstore requis - installation directe depuis le navigateur
- ✅ Mises à jour automatiques
- ✅ Accès aux notifications (notifications push)
- ✅ Ecran de démarrage personnalisé
- ✅ Barre de statut personnalisée
- ✅ Interface adaptée au mobile (fullscreen)

## 📲 Comment installer l'app sur Android

### Option 1: Installation via le navigateur (Recommandé)

#### Sur Chrome / Chromium
1. Allez sur: **https://sport-alpha-lake.vercel.app**
2. Attendez quelques secondes que la page se charge
3. Vous noticerez une barre en bas disant **"Installer 'Sport'"** ou un bouton **"+"** en haut à droite
4. Cliquez sur **"Ajouter à l'écran d'accueil"** ou le bouton **"Installer"**
5. Confirmez l'installation
6. L'application apparaîtra sur votre écran d'accueil! 🎉

#### Sur Firefox
1. Allez sur: **https://sport-alpha-lake.vercel.app**
2. Appuyez sur le bouton menu (3 points) en haut à droite
3. Sélectionnez **"Ajouter à l'écran d'accueil"**
4. Confirmez et c'est prêt!

#### Sur Edge
1. Allez sur: **https://sport-alpha-lake.vercel.app**
2. Appuyez sur l'icône "..." en haut à droite
3. Cliquez sur **"Installer cette application"**
4. Confirmez l'installation

#### Sur Samsung Internet
1. Allez sur: **https://sport-alpha-lake.vercel.app**
2. Appuyez sur le bouton menu
3. Cliquez sur **"Ajouter à la page d'accueil"**
4. Confirmez

### Option 2: Installation manuelle (Mode d'accès rapide)

Si le bouton d'installation n'apparaît pas automatiquement:

1. Ouvrez le navigateur et allez sur: **https://sport-alpha-lake.vercel.app**
2. Appuyez le bouton **menu** (⋮) de votre navigateur
3. Cherchez **"Ajouter à l'écran d'accueil"** ou **"Installer l'app"**
4. Confirmez l'installation

## 🚀 Utilisation de l'app

Une fois installée, vous pouvez:

- **Lancer l'app** comme n'importe quelle application Android
- **Accéder à l'app même hors ligne** - les pages déjà visitées seront disponibles
- **Recevoir des notifications** (si activées)
- **Utiliser les raccourcis**:
  - Appuyez longtemps sur l'icône de l'app pour voir les actions rapides
  - Entraînement, Mini Jeux, Social

## 🔄 Mises à jour

L'application se met à jour automatiquement quand vous êtes en ligne. Vous pouvez forcer une mise à jour en:

1. Fermant complètement l'app
2. La relançant
3. L'app vérify si des mises à jour sont disponibles

## 📊 Authentification

Votre session et vos données sont sécurisées via:
- **JWT tokens** stockés localement
- **HTTPS** pour toutes les communications
- **Authentification** requise pour les actions sensibles

## 🛟 Dépannage

### L'app ne s'installe pas
- Vérifiez que vous utilisez une version récente de votre navigateur
- Essayez avec un autre navigateur
- Assurez-vous que le HTTPS fonctionne correctement

### L'app ne fonctionne pas hors ligne
- Assurez-vous que vous avez visité les pages en ligne d'abord
- Les Service Workers mettent en cache les données en visite
- Seules les pages visitées seront disponibles hors ligne

### Supprimer l'app
- Appuyez longtemps sur l'icône de l'app
- Sélectionnez **"Supprimer"** ou **"Désinstaller"**

## 🔐 Sécurité et Confidentialité

- Toutes les données sont synchronisées via API sécurisée HTTPS
- Votre JWT token est stocké localement et jamais partagé
- Vous pouvez vous déconnecter à tout moment
- Vos données sont conformes à la protection RGPD
